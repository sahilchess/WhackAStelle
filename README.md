## wut this?
stelle pop up randomly across the board and you gotta whack them before they duck back down.

## Features
* random MAXSTELLAR spawn timing
* score tracking

## How to test it
1. click the [live demo link](sahilchess.itch.io/whack-a-max/) at the top of this file
2. click start
3. whack them MAXSTELLAR moles as they pop up
4. check your score at the end
